MajorMud - Reforged Documentation

Welcome to the official public documentation repository for MajorMud - Reforged.

This repository is dedicated to documenting the design, research, and long-term vision of the Reforged project while encouraging community participation.

Project Goals

- Preserve the original MajorMUD gameplay experience.
- Modernize the engine while remaining faithful to the original.
- Build a data-driven architecture.
- Document every significant design decision.
- Preserve historical MajorMUD knowledge and tools.
- Encourage community feedback and suggestions.

Current Planned Features

- Modern server engine
- Data-driven quest system
- Discovery-based world map
- Integrated in-game Explorer
- Multi-character launcher (up to 3 simultaneous characters)
- World editor
- Quest editor
- Item editor
- Monster editor
- Improved UI
- Plugin support
- AI-assisted tools

Core Design Principles

- Original gameplay comes first.
- Verify information before implementation.
- Preserve historical knowledge.
- Document all major decisions.
- Keep the project maintainable for the future.

Community feedback is welcome through GitHub Discussions and Issues.
